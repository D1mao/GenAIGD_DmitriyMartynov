# ControlNet – подготовка

## Что хотим контролировать
- Позу персонажа (чтобы удерживать одинаковую позу для всех четырёх гоблинов).
- Глубину сцены (для локаций: чтобы пещера не выглядела плоской).
- Контур сундука / трофея (чтобы форма предмета не плыла при inpaint).

## Выбранные типы ControlNet

### Pose (OpenPose)
- **Зачем:** хотим сделать серию кадров, где все гоблины чихают в одной позе. OpenPED сохранит кости и ориентацию тела.
- **Что без него плывёт:** руки и углы плеч каждый раз разные.
- **Референс:** скетч позы (приложу рисунок от руки).

### Depth
- **Зачем:** для пещерных фонов, чтобы создать ощущение глубины и масштаба.
- **Что без него плывёт:** стены сливаются в плоскую текстуру.

### Canny
- **Зачем:** для сундука и других трофеев – чтобы края были чёткими и не размывались при inpaint.
- **Референс:** контур сундука из референса локации.

## Ориентиры (рефы)
1. **Референс глубины пещеры:** кадр из мультфильма «Тайна Коко» – подземный мир с ярусами. Берём мягкий переход планов.
2. **Контур сундука:** референс из игры «Zelda: Breath of the Wild» – массивный деревянный сундук с металлическими уголками.
